# Travel Memory Assets

This pack contains the first verified visual batch for the forest, beach, and star-garden travel scenes.

- Source: Agnes `agnes-image-2.1-flash`
- Runtime metadata: `manifest.json`
- Prompts and raw responses: `reference/`
- Transparent floating assets: `badges/`, `fridge-magnets/`
- Opaque card surfaces: `cards/`, `pet-cards/`

The webpage keeps HTML text over the images. If an image fails to load, the runtime falls back to the scene emoji and CSS card.
