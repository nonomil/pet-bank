# UI 优化 v2：自然场景绘本风

> 在 v1 童趣玻璃态基础上，参考「汉字游乐场」博文界面（手绘扁平 + 自然场景 + 白底卡），统一优化 4 界面。
> 日期：2026-07-03 · 状态：方案已定（用户拍板 3 决策），执行中。

## 背景

v1（commit 4e9dda3 / 916d05b）已落地「Agnes 抽象渐变背景 + 玻璃态卡 + emoji + 童趣字体」。
用户提供参考案例博文截图（图1 为主：手绘扁平汉字配对游戏，天空+草地+河流+花，白底卡，字+拼音+插图，熊猫吉祥物），要求 4 界面都往该风格优化。

## 参考案例视觉要素（子代理提取）

- 背景：自然场景（天空蓝 + 草地绿 + 河流蓝 + 远山 + 花）
- 卡片：白底 + 圆角 10-20px + 轻阴影 + 半透明中央容器
- 插图：手绘扁平卡通（粗描边 + 平涂），动物/植物/物品，高饱和
- 字体：无衬线圆体（PingFang/Noto Sans），汉字加粗
- 强调色：橙(计时)/黄(星)/红(错)/青(进度)
- 通关弹窗：统计小卡(浅绿/粉/橙) + 胶囊按钮 + 吉祥物

## 当前现状 6 个不一致（子代理摸底）

圆角 6 档(13/14/16/18/20/24) · blur 不齐(8/10/12) · border 不一致 · 兜底色 3 套 · 强调色分裂(多彩/黄绿/金) · 装饰密度不均。

## 3 个决策（用户拍板）

1. **背景** → 重生自然场景（Agnes 生 3 张）
2. **插图** → 筛字生手绘插图（有画面感的 HSK1 字，字+拼音+图）
3. **卡片** → 白底偏白 + 轻微玻璃（白 0.88 + blur 6px）

## 执行清单

### P1 背景（Agnes，低风险）
- `playground-bg.png`：草原出发地（远山+蓝天+草地野花+小路/气球，冒险入口感）
- `hanzi-bg.png`：森林花田（树木+花丛+小溪+阳光斑驳，宁静学习感）—— 大厅+答题共用

### P2 字插图（Agnes，中风险）
- 筛 HSK1 有画面感字（自然物+动物为主，**规避人物/抽象词**——E5 验证人物词如「孩子」会跑偏成花）
- 候选 ~30 字：山 水 日 月 花 木 火 雨 雪 云 石 田 土 鸟 马 牛 羊 鱼 虫 树 草 果 瓜 风 星…
- 复用 `scripts/generators/gen_hanzi_images.py`（E5 已验证）
- 命名 `assets/ui/hanzi-img/{pinyin}.png`
- 题卡渲染：`hanzi-game.js` 有图字用 `<img>`，无图字 fallback emoji

### P0+P3 CSS 改造（纯 CSS，0 风险）
- 白底偏白：`rgba(255,255,255,0.88)` + `blur(6px)`（替 0.55+10）
- 圆角节奏：小卡 10 / 中卡 14 / 大卡 20 / 胶囊 24
- border 统一 2.5px 主题色描边
- 兜底色统一（自然场景主色，如 `#E6F4F1`）
- 强调色调色板：橙 `#FFA500` / 黄 `#FFD700` / 红 `#FF6B6B` / 青 `#20B2AA` / 绿 `#4CAF50`
- 装饰均衡：排行榜补自然元素，playground 减密
- 文件：`css/playground.css` + `css/hanzi-game.css` + `css/leaderboard.css`

## 风险

- 内容图准确率（E5：人物词跑偏）→ 筛字规避，跑偏字 fallback emoji
- 生图耗时（背景 3 + 插图 ~30，并发 3-5 路约 30-60 min）
- 体积（插图 PNG 偏大，后续 pngquant/webp）

## 验收

- 4 界面真机截图，视觉接近参考案例（自然场景 + 白底卡 + 插图 + 收敛色）
- 参数一致（圆角/blur/border/兜底色/强调色 统一）
- 无功能回归（玩法/排行榜/记忆系统照常）
