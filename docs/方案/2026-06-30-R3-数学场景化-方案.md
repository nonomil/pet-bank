## 1. 问题

当前 math 事件题面是 genMathQuestion 生成的**裸算式**（`3 + 5 = ?`）或 CMATH 随机应用题（"妈妈包饺子"），event.text 只是"举牌子/出题"的包装引导句。离 [05_事件规则](../叙事重构/05_事件规则.md)「题目与场景语义一致」有距离——孩子看到的是通用算式/通用情境，不是"森林蘑菇""瀑布露珠"的场景题。

章节有视觉分组（5 章着色）但数据层无"本章练什么"约束。

## 2. 方案

### 2.1 核心机制：math 事件可选固定场景题

math 事件新增可选字段 `question`/`answer`/`options`（固定场景题）。`genMathQuestion` 优先用 event 自带题，否则回退现有随机逻辑（arithmetic/word-CMATH/logic）。

```js
// genMathQuestion 小改（exploration-detail.js）
function genMathQuestion(mathType, difficulty, event) {
    // 优先：事件自带固定场景题（R3 场景化）
    if (event && event.question && event.answer != null) {
        return { text: event.question, answer: event.answer, options: event.options || genMathOptions(event.answer) };
    }
    // 回退：现有随机逻辑（arithmetic/word-CMATH/logic）
    ...
}
```

调用处（showNextEvent math 分支）传 event。**改动量：genMathQuestion 加 event 参数 + 优先判断（~5 行）**。

### 2.4 数据结构（chapter_skill 在场景顶层，math 事件只加题字段）

> codex 二审阻塞项：chapter_skill 必须在场景 JSON 顶层（非 math 事件内），便于按场景审查/配图/统计。

**场景顶层**（data/stories/forest.json）新增 chapter_skill：

### 2.5 渲染（showNextEvent math 分支，复用现有拼接，仅扩判断）

现状 L349-351 拼接 `event.text + <br> + q.text`，按 `mathType==='word'` 选 `.galgame-word`（17px 无 letter-spacing）否则 `.galgame-math`（22px letter-spacing）。

**长场景题面用 `.galgame-word`**（避免 letter-spacing 错乱）。判断改为：

```js
const useWordStyle = event.question || event.mathType === 'word';
const qHtml = `${event.text}<br><span class="${useWordStyle ? 'galgame-word' : 'galgame-math'}">${q.text}</span>`;
```

（复用现有拼接逻辑，只扩 useWordStyle 判断；.galgame-word/.galgame-math 样式不动。）

### 3.1 实施备注（codex 二审落实）

- **事件 id 不改**：保留 `forest.math2` 等现有 id，只加字段不改键名（配音映射/审查引用/diff 成本）。
- **word 回退依赖 CMATH 池**：无 question 的 word 事件，CMATH 池可用时抽 CMATH 应用题，池不可用时落回算式（与当前一致，非新回归）。
- **渲染约束**：`useWordStyle = !!event.question || event.mathType === 'word'`；固定长题面一律 `.galgame-word`，仅裸算式保留 `.galgame-math`。
- **配音映射执行项**：math 文本改后，新题面需用 voxCPM2 生成 mp3 + 登记 assets/voice/map.json（文本 md5→mp3）；id 不变降低映射漂移。本批先改文本，配音补录作为后续。

## 4. 验收

1. 12 场景 math 显示**场景题面**（蘑菇/露珠/糖果…），非裸算式。
2. 每场景 json 含 chapter_skill；README 含 5 章教育目标表。
3. 答对/答错逻辑不变（answerMath）；选项 4 个含正确答案。
4. 回退正常：若某 math 事件无 question（未来新增），genMathQuestion 回退随机。
5. 不回归：路线地图/章节/怪物/战斗/语音/CMATH 题库。

## 5. 风险

- 低：数据改动为主，genMathQuestion 小改（加参数 + 优先判断），回退保证零回归。
- 题面文案需儿童可读 + 配音友好（短句、数字读法）。

## 6. 不做（边界）

- 不加新场景、不改引擎核心、不改 CMATH 题库、不引入分支系统。
- R4（遭遇/旁白）R5（死代码）本批不做，留后续批次。
