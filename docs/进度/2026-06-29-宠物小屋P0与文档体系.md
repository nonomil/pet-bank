### 2.5 MC 宠物图修复（cv2 本地批处理）

**诊断**：
- `assets/pets/poses/mc_*.png` 共 **40 张**，实为「多体 + 箭头」进化链对比图（**34 张多体** + **6 张 ultimate 单体**），作为单体立绘不合适（含多个形态并列 + 指示进化的箭头）。

**方案**（纯本地 cv2，无需外部 API）：
1. 形态学去箭头：椭圆核开运算剥离细线状箭头。
2. 最大连通块：保留主体宠物，丢弃箭头与小噪点。
3. 裁切居中：按主体外接框居中裁切。
4. Lanczos 插值放大到 1024²（高质量重采样）。

**验证**：
- playwright 视觉抽样 **5 张**（enderman / wolf / cat / parrot / turtle，跨阶段）**全 PASS**：1 个体 / 完整 / 无箭头 / 居中。

**说明**：
- Agnes API key（用户提供）返回 **401 无效**，未用；改走纯本地 cv2 方案。
- 原图备份在 `.tmp/mc-backup/`，git v0.3.0 tag 也有原版可恢复。

**涉及文件**：
- `assets/pets/poses/mc_*.png`（40 张重写）
- `.tmp/`（cv2 处理脚本 + 备份）
