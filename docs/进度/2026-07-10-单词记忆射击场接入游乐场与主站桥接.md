# 2026-07-10-单词记忆射击场接入游乐场与主站桥接

> 对应版本：`v0.7.22`

## 1. 本轮目标

把 `prj/单词记忆射击场原型` 以“主站子页内嵌 + 还能独立打开”的混合方式接入 `游乐场`，同时补上最轻量的一层主站成长桥接：

- 不重做原型结构
- 不改词库、主角选择、关卡和投掷逻辑
- 只在通关结算时回写主站积分与摘要

## 2. 已完成内容

### 2.1 主站游乐场接入

- `index.html` 新增 `单词记忆射击场` 特色入口卡
- 新增 `page-word-memory-map` 子页宿主壳
- 保留 `独立打开` 入口，继续直达原型页
- 新增静态深链入口 `app/playground/word-memory-map/index.html`

### 2.2 路由与壳层收口

- `js/app.js` 新增：
  - `word-memory-map` 的 `PAGE_TO_TAB`
  - `/app/playground/word-memory-map` canonical route
  - `/playground/word-memory-map` legacy alias
  - `ensureWordMemoryMapEmbed()`
- `index.html` 新增 `?route=` 恢复脚本，解决直接访问深层子页时的 404 问题
- `index.html` 挂上 `css/playground.css`，让 iframe 宿主页和其他游乐场子页保持同一套样式

### 2.3 主站成长桥接

- `prj/单词记忆射击场原型/game.js` 在通关结算 `finishGame()` 时发送 `postMessage`
- 回传字段包含：
  - `score`
  - `earnedStars`
  - `accuracy`
  - `levelOrder`
  - `levelTitle`
  - `highestUnlockedLevel`
  - `heroId`
  - `worldPack`
- 主站 `js/app.js` 新增 `petbank_word_memory_map_progress`
- 子页 summary 面板新增：
  - 累计通关
  - 累计星星
  - 累计成长分
  - 最佳命中率
  - 最高解锁
  - 最近通关
  - 最近主角
  - 最近地图
