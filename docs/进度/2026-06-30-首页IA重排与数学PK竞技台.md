## 1. 首页 / 积分 信息架构重排

### 1.1 痛点
v0.5.3 之前，`成长地图`(map) 同时充当「首页总览」和「积分入口」——路线图、showcase、积分 hub(今日打卡/每周复盘/数学PK/设置) 全压在一页，**首页与积分混在一起**。

### 1.2 方案（5 tab）
| tab | 落地 | 收纳 |
|---|---|---|
| 🏠 首页（默认） | page-map 改造的 dashboard | hero 摘要 + **6 张大卡片启动器** + 宝箱仓库 |
| ⭐ 积分 | page-today | 今日打卡 + 每周复盘/数学PK/奖励兑换/兑换商店/背包（兑换并入积分） |
| 🐾 宠物 | page-pet | 小屋 / 图鉴 / 遛弯 |
| 🗺️ 探索 | page-explore | 成长地图(路线图) + 场景预览 + 战斗 |
| ⋯ 更多 | page-works | 工具箱 / 设置 |

- 首页大卡片一键跳转，**不必点顶部 tab**（`dash-card` 样式，移动端 2 列）。
- `js/app.js`：`HOME_TAB_MAP` / `PAGE_TO_TAB` 重写（兑换→积分、settings→更多、map→首页）。

### 1.3 文件
- `index.html`（nav 重排 + page-map dashboard 化 + page-today/works hub-nav）
- `js/app.js`（路由映射）
- `css/style.css`（`.dash-card` 大卡片样式）

## 2. 数学 PK 全屏人机竞技台

### 2.1 设计
- **全屏竞技台**（进入数学 PK 即全屏，覆盖顶栏/侧栏），竞技台背景 + 左人/右机器人立绘 + VS，**不复用旧卡片**。
- **同题竞速**：每轮同一题，机器人按难度有思考倒计时条；人在机器人之前答对 → 赢该轮；答错不结束、继续抢答。共 **5 轮**。
- 数字键盘（1-9 + 清除/0/确认）+ 计算器风显示屏 + 结果反馈；物理键盘支持。
- 难度在「设置」页选（简单/中等/困难），中等档约 30% 出 CMATH 应用题（grade 2）。
- 计分：赢一轮 +10 +连击加成，赢整局 +25，结束回写成长积分（保留最高分）。
