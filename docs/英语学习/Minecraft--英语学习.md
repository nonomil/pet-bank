# Minecraft 我的世界英语学习二期方案

> 定位：在当前已经落地的 `english-mc-hybrid-2026` 英语混合资料包上继续增强，不另起新系统。当前项目继续负责入口、计划、测验、进度、积分和奖励；外部站继续负责点读、绘本阅读和词汇素材启发。

## 1. 背景

当前可参考的外部入口：

- 绘本 / 点读站：https://mayihaoke.com/
- Minecraft 5-6 岁绘本阅读：https://mayihaoke.com/mcbook56/read
- Minecraft 单词学习：https://mayihaoke.com/minewords

现有项目里已经有 `Minecraft我的世界英语故事` 资料包：

- 资料包 id：`english-mc-hybrid-2026`
- 当前模式：混合式英语学习
- 已有模块：`mcbook56-story`、`mcbookstarters-reader`、`english-weekly-review`、`custom-entry-slot`
- 已有奖励：英语章节 `+2`，周复盘 `+5`，连续 3 个英语章节额外 `+1`

所以这次不做“重新接入英语学习”，而是做“英语学习理解增强”。核心问题是：孩子看完画面、听完英语以后，可能只是记住了图片或熟悉了声音，并没有真正建立英文、中文、情境和可回忆表达之间的关系。

## 2. 目标

把当前英语包从“打开外部阅读页 -> 回来打勾”升级为：

1. 看故事前知道本节目标词。
2. 看故事时保留外部点读页的轻量体验。
3. 看故事后完成 2-3 道轻测验，确认不是只看过画面。
4. 从故事里沉淀 Minecraft 单词卡，按级别和难度复习。
5. 掌握单词后获得英语星星、碎片或兑换券，再兑换 Minecraft 主题宠物卡片。

最终要解决的是“能不能认出来、想起来、用一点点”，而不是只记录“今天打开过页面”。

## 3. 产品边界

### 本期做

- 在现有英语资料包内新增“章节轻测验”。
- 增加 Minecraft 单词卡模块。
- 增加单词掌握状态：`new`、`learning`、`mastered`。
- 增加英语专属奖励媒介：英语星星 / 词卡碎片 / 兑换券。

### 本期不做

- 不新建第二套积分系统。
- 不让学习单额外发管理奖励。
- 不一次做复杂的自定义后台或开放上传系统。

## 4. 推荐学习流程

### 绘本章节

1. 孩子在学习中心打开 `Minecraft我的世界英语故事`。
2. Lesson 页显示本节目标词、预计时长、家长提示。
3. 点击“打开点读页”，跳到外部绘本阅读页。
4. 听读完成后回到当前项目。
5. 先做 2-3 道轻测验。
6. 测验通过后点“读完打勾”，发放本节英语章节奖励。
7. 如果累计达到连续学习里程点，继续触发已有连读奖励。

### 单词卡

1. 孩子进入“Minecraft 单词卡”模块。
2. 先看图和例句，不要求死背。
3. 通过选择题、图文匹配、英文选中文来复习。
4. 连续答对后变成 `mastered`。
5. 掌握一定数量后获得兑换券或卡片碎片。

## 5. 章节测验设计

测验目标不是考试，而是让孩子从“被动看过”转成“主动回忆一次”。

### P0 题型

| 题型 | 说明 | 示例 |
|---|---|---|
| 英文选中文 | 给英文句子，选中文意思 | `I see a block.` -> `我看见一个方块。` |
| 中文选英文 | 给中文意思，选英文句子 | `我有一把剑。` -> `I have a sword.` |
| 单词识别 | 给英文单词，选图片或中文 | `door` -> `门` |
| 图片选词 | 给图片，选对应英文词 | 方块图 -> `block` |

### P1 题型

- 句子排序：`I / see / a / block` 排成 `I see a block.`
- 听音选择：播放单词或短句，选对应英文。
- 缺词补空：`I see a ____.` 选择 `block`。

### 通过规则

- 每节 3 题。
- 答对 2 题即可通过。
- 未通过可以重做，但不重复发积分。
- 错题要显示简短解释，避免只告诉孩子“错了”。
- Lesson 奖励仍沿用现有规则，建议测验通过后才允许完成打勾。

## 6. 单词卡设计

### 单词卡字段

建议新增 `minecraft-vocab` 模块，每张卡包含：

```json
{
  "id": "mc-word-block",
  "word": "block",
  "translation": "方块",
  "level": "starter",
  "difficulty": 1,
  "sourceModuleId": "mcbook56-story",
  "sourceLessonId": "chapter-01",
  "example": "I see a block.",
  "exampleZh": "我看见一个方块。",
  "image": "assets/learn/english/minecraft-vocab/block.webp",
  "tags": ["minecraft", "object", "starter"],
  "distractors": ["door", "stone", "sword"]
}
```

### 难度分层

| 层级 | 内容 | 目标 |
|---|---|---|
| Starter | 常见名词、颜色、动作 | 能看图认词 |
| KET-1 | 简单短语、常用动词 | 能英文选中文 |
| KET-2 | 简短句子、情境表达 | 能理解一句话 |
| Review | 错题和遗忘词 | 间隔复习 |

### 掌握状态

- `new`：未学或只看过。
- `learning`：答错过，或只答对 1 次。
- `mastered`：连续答对 2 次。

答错后不直接清零，可以把 `streak` 归 0，并增加 `wrong` 次数。复习时优先抽 `wrong > 0` 的词，再抽新词，最后抽已掌握词做少量保持。

## 7. 奖励与兑换

### 原有成长分保持稳定

英语主奖励继续按 lesson 发：

- 完成章节测验并打勾：沿用 `+2`
- 每周英语复盘：沿用 `+5`
- 连续 3 个英语章节：沿用额外 `+1`

单词卡不建议直接大量发成长分，否则会冲击宠物成长节奏。

### 新增英语专属奖励

建议新增轻量媒介：

| 奖励 | 来源 | 用途 |
|---|---|---|
| 英语星星 | 完成单词练习、复习错词 | 展示英语努力值 |
| 词卡碎片 | 掌握指定数量单词 | 合成 Minecraft 卡片 |
| 英语兑换券 | 完成阶段目标 | 兑换宠物卡片或主题装饰 |

### 兑换建议

- 掌握 5 个词：获得 1 个词卡碎片。
- 掌握 10 个词：获得 1 张普通 Minecraft 主题卡片。
- 掌握 20 个词：获得 1 次稀有卡片兑换机会。
- 完成 1 周英语复盘：可额外获得 1 个英语兑换券。

兑换奖励要写入现有背包 / 卡片体系，避免另做一套奖励库存。

## 8. 素材策略

外部站点可以作为学习入口和内容启发，但本项目不应默认批量复制第三方素材。

推荐顺序：

1. 先手工整理 20-40 个高频词，做 MVP。
2. 图片优先使用项目内已有 Minecraft 风格宠物或自生成图片。
3. 如果后续确认素材授权，再做 importer。
4. importer 只保存最小元数据：来源 URL、章节、词、图片引用、更新时间。
5. 所有外部来源数据都加 `sourceProvider` 和 `sourceUrl`，方便日后审计。

## 11. 阶段路线

### Phase 1：数据与文档

- 完成本方案和实施计划。
- 新增 20-40 个种子词。
- 给 3 个现有英语章节补 `quiz` 示例。
- 增加数据合同测试。

### Phase 2：章节测验

- LearnCenter 支持渲染 `lesson.quiz`。
- 通过测验后才允许完成打勾。
- 完成状态、防重复奖励继续复用现有逻辑。

### Phase 3：单词卡

- 新增 `minecraft-vocab` 模块。

### Phase 4：兑换奖励

- 新增英语星星 / 碎片 / 兑换券记录。
- 接入宠物卡片或卡片碎片兑换。
- 增加防重复兑换规则。

### Phase 5：素材增强

- 评估外部素材授权。
- 必要时做 importer。
- 图片本地化、压缩和来源审计。

## 12. 验收标准

1. 英语章节页面能显示外部阅读入口、目标词和测验。
2. 测验未通过时不能领取本节奖励。
3. 测验通过后完成 lesson，仍只发一次成长分。
4. 单词卡能按 `new / learning / mastered` 记录状态。
5. 答错词会进入优先复习。
6. 掌握指定数量单词后能获得兑换媒介。
7. 兑换宠物卡片或碎片不会重复领取。
8. 多孩子切换后，英语测验、词卡和兑换记录相互隔离。
9. 学习单仍只负责记录和收尾，不额外发管理奖励。
10. 第三方素材没有被无边界批量复制进仓库。

## 13. 推荐下一步

优先做一个小闭环：

1. 给 `mcbook56-story` 的 3 个章节补测验。
2. 新建 `minecraft-vocab`，先放 20 个词。
3. 做一个极简单词卡练习页。
4. 掌握 10 个词后给 1 张普通 Minecraft 主题卡片。

这个闭环够小，但能验证最关键的产品假设：孩子是否真的从“看过故事”变成“能认出几个词、能完成一次主动回忆”。
