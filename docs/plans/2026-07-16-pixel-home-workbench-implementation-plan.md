# 像素成长工作台首页实施计划

> **给 Claude:** 必需子技能：使用 superpowers:executing-plans 来逐任务实施此计划。

**目标：** 将独立像素首页 Demo 改造成参考截图布局、沿用现有明亮像素美术的三栏成长工作台。

**架构：** 保持单文件静态 Demo，不新增构建工具或主站依赖。HTML 负责语义结构，内联 CSS 负责像素工作台视觉与响应式布局，内联 JavaScript 仅维护当前栏目、当前世界和 Toast 反馈。

**技术栈：** Vanilla HTML、CSS Grid/Flex、原生 JavaScript、本地 WebP/PNG 素材。

---

### 任务 1：建立三栏工作台骨架

**文件：**
- 修改：`tmp/pixel-home-template-demo/index.html`

**步骤 1：保留独立 Demo 原有相对资源路径和页面入口。**

**步骤 2：把主体调整为顶部栏下的工作台网格。**

桌面布局使用 `grid-template-columns: 220px minmax(0, 1fr) 280px`；中间区最小宽度为 0，避免长标题撑出横向滚动。

**步骤 3：把原左侧今日任务转换为纵向栏目面板。**

栏目包含学习总览、今日任务、幼小衔接、绘本书架、汉字拼音、数学启蒙、英语故事、学习游戏、资源地图、成长成就，并保留一个小型陪伴状态区。

**步骤 4：把原主图区域转换为中间主线面板。**

主线面板继续使用森林地图作为默认图片，保留森林/星际/方块世界的动态切换入口。

**步骤 5：把原成长卡片和快捷入口合并为右侧进度面板。**

右侧按“任务面板、下一步要做什么、最近获得”分区，保证截图参考的工作台信息密度。

### 任务 2：实现中心三列内容卡片

**文件：**
- 修改：`tmp/pixel-home-template-demo/index.html`

**步骤 1：在主线面板下增加 6 张内容卡片。**

卡片覆盖幼小衔接、绘本书架、汉字拼音、数学启蒙、英语故事、单词远征；每张卡片固定图片比例、标题、说明、进度条和入口动作。

**步骤 2：将三个世界入口保留为独立选择区。**

世界入口使用现有 `forest.webp`、`sci-fi.webp`、`block.webp`，并通过 selected 状态表达当前世界，不新增大尺寸生成图。

**步骤 3：替换随意字符图标。**

使用 CSS 几何形状和语义标签制作统一小图标，图标不承担唯一信息，文字仍保留。

### 任务 3：补齐交互和可访问性状态

**文件：**
- 修改：`tmp/pixel-home-template-demo/index.html`

**步骤 1：为左侧栏目绑定选中态和 Toast。**

点击栏目只更新 Demo 状态，不写入 `localStorage`，避免污染主站孩子数据。

**步骤 2：扩展世界切换。**

锁定入口只展示锁定信息，不弹出气泡；减少动画偏好时禁用位移和渐变动画。
