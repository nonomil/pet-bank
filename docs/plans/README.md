# 设计与实施计划索引

> 本目录存放**下一阶段准备执行**的正式设计稿与实施计划。
> 和 `docs/方案/` 的区别是：`方案/` 更偏历史方案与变更归档，`plans/` 更偏“接下来要怎么做”。

---

## 1. 编号规则

- `DESIGN-*`：设计稿，冻结产品边界、交互和数据模型
- `PLAN-*`：实施计划，写清楚具体改哪些文件、按什么顺序做、如何验证
