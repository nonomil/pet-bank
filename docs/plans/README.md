## 3. 阅读建议

- 想看“为什么这样设计”：
  先读 `DESIGN-*`
- 想看“真正要改哪些文件、怎么落地”：
  再读 `PLAN-*`
- 想看“这一轮已经做了什么”：
  回到 [../进度/README.md](../进度/README.md)
- 想看“为什么范围变了、之前发生了什么”：
  回到 [../方案/README.md](../方案/README.md)
