## 2. 题型体系（4 大类）

### 2.1 基础运算（arithmetic）— 已实现
| 难度 | 运算 | 范围 |
|------|------|------|
| easy | + - | 5~25 / 15~40 |
| medium | × | 2~9 |
| hard | × ÷ | 2~9（整除） |

### 2.2 应用题（word）— 新增
情境化、场景相关，培养"数学建模"。题干 = 故事，答案 = 数字。
| 场景 | 示例 |
|------|------|
| forest | "森林里有 5 朵蘑菇，下雨后又长出 3 朵，现在共有几朵？"（加，答 8）|
| beach | "你捡了 12 个贝壳，送给妹妹 4 个，还剩几个？"（减，答 8）|
| candy | "糖果树有 3 排棒棒糖，每排 4 根，一共几根？"（乘，答 12）|
| space | "15 颗星星平均分给 3 个星座，每个星座几颗？"（除，答 5）|
| volcano | "火山口每天喷 3 颗火水晶，5 天后共几颗？"（乘，答 15）|

### 2.3 数独（sudoku）— 新增（高级）
4×4 简易数独（适合孩子，1-4 每行/列/宫不重复），填空 1-2 格。
```
1 _ | 3 4
3 4 | _ 2
---------
_ 3 | 4 1
4 1 | 2 _
```
答：填缺失格（如第1行第2列=2）。后段场景（space/stargarden）出现。

### 2.4 逻辑题（logic）— 新增（高级）
- **找规律**：2, 4, 6, ? （答 8）／1, 1, 2, 3, 5, ? （斐波那契，答 8）
- **等式平衡**：? + 3 = 10 - 2 （答 5）
- **图形计数**：图中几个三角形

## 3. 难度分级（场景 × 宠物等级）

| 阶段 | 场景 | 题型 | 难度 |
|------|------|------|------|
| 前段 | forest / beach / waterfall | 基础加减 + 简单应用题 | easy |
| 中段 | mountain / candy / castle / desert | 基础乘除 + 应用题 | medium |
| 后段 | space / volcano / stargarden / underwater / cave | 乘除 + 应用题 + 数独/逻辑 | hard |

**动态调节**（可选）：宠物 Lv.越高，数字范围越大（Lv.1-3 用 easy 范围，Lv.4+ 用 medium，Lv.7+ 用 hard）。

## 4. 奖励机制

| 结果 | 奖励 |
|------|------|
| 答对 | exp +15~30（难度递增）+ 概率 item + 解锁路径/宝箱 |
| 答错 | 扣 HP 5~10（轻度惩罚）或触发战斗（failAction: 'battle'）|
| 连续答对（连击）| 额外 exp 奖励（连击 ×1.5）|

> 与数学 PK Tab（连击练习）共享出题逻辑，但探索解谜是"融入式单题"，PK Tab 是"连续练习"。

## 6. 出题实现（genMathQuestion 扩展）

```js
function genMathQuestion(type, difficulty) {
  // type: 'arithmetic' | 'word' | 'sudoku' | 'logic'
  switch(type) {
    case 'arithmetic': return genArithmetic(difficulty);  // 现有
    case 'word': return genWordProblem(currentScene.id, difficulty);  // 应用题库
    case 'sudoku': return genSudoku4x4();  // 4x4 数独填空
    case 'logic': return genLogic();  // 找规律/等式
  }
}
```

- 应用题库：`data/math-word.json`（每场景几道，情境化）
- 数独：`genSudoku4x4()` 生成 4x4（挖 1-2 格），4 选项
- 逻辑：找规律/等式模板

## 8. 优先级

1. **P1 扩基础运算**：beach/waterfall/mountain/candy 加 easy/medium arithmetic（复用 genMathQuestion）
2. **P1 应用题**：建 `data/math-word.json` + genWordProblem（场景情境化）
3. **P2 数独/逻辑**：genSudoku4x4 + genLogic（后段场景）
4. **P2 难度动态调节**：按宠物等级
5. **P3 连击奖励** + 失败惩罚平衡

---

## 9. 边界

- 数独只做 4x4（孩子向，不做 9x9）
- 应用题数字小（< 50），情境贴近场景
- 不引入负数/分数/代数（保持小学难度）
- 探索解谜与数学 PK Tab 共存（不合并）
