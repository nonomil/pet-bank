# 学习中心中文资料包封面图设计稿

> 日期：2026-07-05
> 范围：仅调整 `幼小衔接暑假中文资料包` 在学习入口大厅里的选项卡封面图

---

## 1. 目标

把当前中文资料包入口从“普通书堆素材”升级成一张更像产品封面的 Agnes 生图。

这张图需要做到：

- 一眼看出是 `幼小衔接暑假中文资料包`
- 更贴近 `暖黄绘本课堂` 的幼小衔接气质
- 放进当前学习入口卡片后仍然好看，不被裁坏
- 和现有英语资料包、学习网站入口包保持同一组卡片封面语言

---

## 2. 选定方向

用户已确认采用：

- 风格：`暖黄绘本课堂`
- 构图：`课桌学习角`

因此新图的主视觉锚点固定为：

- 木课桌
- 晨光窗边
- 拼音卡
- 田字格
- 古诗小卷轴
- 小书包
- 铅笔与中文绘本

并明确不加入人物，避免年龄感、面部风格和手势跑偏。

---

## 3. 画面要求

### 3.1 气质

整体应像儿童绘本扉页或启蒙资料包封面，而不是单个静物素材。

关键词：

- 温暖
- 安静
- 晨读感
- 适合 5-6 岁
- 中文启蒙

### 3.2 构图

当前卡片图片区会以 `object-fit: cover` 方式放进学习入口卡。

因此新图必须优先适配：

- 横向卡片封面
- 主体集中在中间偏下
- 底部预留较干净区域，避免和 `晨读 / 古诗 / 识字` 徽章打架
- 画面四周可允许少量环境延展，便于裁切

### 3.3 禁止项

- 不要人物
- 不要真实照片感
- 不要复杂教室全景
- 不要密集文字
- 不要真的出现中文、拼音字母或可读标题
- 不要 App UI、按钮、图标、贴纸式大字

---

## 4. 接入策略

不替换整套学习页布局，只替换中文资料包入口卡的图片资源。

落地方式：

1. 用 Agnes 生成新的中文封面图
2. 保存到 `assets/learn/`
3. 把 `js/learn-center.js` 中中文卡的 `imageSrc` 从 `assets/decor/book_stack.webp` 切到新图
4. 视实际裁切效果微调 `imageStyle`
5. 在 smoke 中增加中文封面路径校验

---

## 5. 验收标准

满足以下几点即可视为完成：

1. 学习首页中文资料包入口不再显示旧书堆图。
2. 新封面一眼能看出“中文启蒙 / 暑假晨读 / 幼小衔接”的气质。
3. 在当前卡片高度下主体不被严重裁切。
4. 与英语封面卡、学习网站入口卡并排时不显突兀。
5. smoke 能校验到新的中文封面路径。
