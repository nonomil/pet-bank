# 像素成长工作台首页设计

## 目标

为独立 Demo 提供一个可审查的像素成长工作台首页：美术沿用现有 Demo 的明亮像素地图与宠物素材，布局参考用户提供的“顶部导航 + 左侧栏目 + 中间卡片 + 右侧进度”截图。

## 范围

- 只修改 `tmp/pixel-home-template-demo/index.html`。
- 不接入主站路由、状态、积分或正式页面。
- 不删除或替换现有故事地图素材；优先复用已存在的森林、星际、方块地图和宠物图。

## 视觉方向

- 主色为明亮森林绿、天空青、暖黄和深蓝墨色，保持儿童产品的清晰度与科技感。
- 顶部导航沿用独立 Demo 的半透明白色圆角矩形，使用轻微玻璃模糊和柔和阴影；栏目按钮用浅绿色选中态形成层次。
- 面板与卡片使用 14-18px 圆角、低对比度细边框、半透明浅色填充和轻玻璃模糊；不使用厚重的 Minecraft 仿制边框、CRT 扫描线或大面积霓虹光晕。
- UI 图标使用统一的 CSS 几何像素图形，不再使用 `✦`、`□` 等随意字符作为功能图标。
- 标题和正文保持高对比度；正文行高不少于 1.5，交互元素保留可见 focus 状态。

## 页面结构

1. 顶部栏：品牌、首页/学习/绘本/探索/游乐场导航、积分与个人入口。
2. 左侧栏目：学习总览、今日任务、幼小衔接、绘本书架、汉字拼音、数学启蒙、英语故事、学习游戏、资源地图、成长成就。
3. 中间工作区：页面标题、今日主线绿色任务面板、三列入口卡片网格；卡片展示地图/绘本/练习图片、短说明、进度和入口按钮。
4. 右侧面板：今日经验、连续学习、故事册数、成长分；下一步行动；最近获得。
5. 移动端：左侧栏目变为可横向滚动的栏目条，中间卡片变为单列，右侧面板移动到卡片下方，禁止页面横向溢出。

## 交互契约

- 点击世界卡片更新当前主线面板的图片、标签、标题、描述和进度。
- 点击左侧栏目更新当前选中态，并显示轻量状态反馈；不跳转正式主站。
- “继续任务”“查看全部”“打开书架”等 Demo 按钮必须有可见反馈。
- 锁定节点只呈现锁定状态，不弹出气泡或遮挡内容。
- 所有按钮具备键盘 focus 样式；减少动画偏好下关闭位移动画。

## 验收标准

- 1440px 桌面视图能同时看到三栏，中心入口卡片每行 3 张。
- 1024px 视图不会挤压正文，右侧面板能合理降级到工作区下方。
- 390px 手机视图无横向滚动，栏目条可滚动，卡片与文字不溢出。
- 森林、星际、方块三个世界都能点击切换主线内容。
- 左侧至少 10 个栏目可见且选中态清楚。
- 页面只依赖本地相对素材，直接通过本地 HTTP 服务打开时无资源 404。
