# 宠物图鉴分馆封面与主题册升级设计

日期：2026-07-04

## 目标

把当前“4 个大色块 + 全量原始系列标签”的图鉴入口，升级成更像图鉴书首页的结构：

- 4 个分馆入口用 Agnes 生成统一风格封面图
- 普通宠物卡全部恢复正常彩色显示
- 收集状态改为“标记式”而不是“禁用式”
- 二级分类从“原始 series 平铺”改成“分馆 -> 主题册”

## 设计结论

### 1. 分馆封面卡

顶层 4 张卡保留，但改成更扁、更像绘本封面的横向图卡。

二次收口要求：

- 封面里的代表卡统一改用新版 `composed-v2` 卡图
- 叠卡主视觉放在卡片右侧区域，不再压在正文中间
- 左侧预留稳定文案区，便于放馆名、简介和进度
- 代表卡可以略微上移，但重点是形成“左文案 / 右叠卡”的稳定结构

- `阳光花园馆`
  - 关键词：阳光花圃、温暖植物、童话花园、金色晨光
- `奇趣冒险馆`
  - 关键词：旅行背包、星盘罗盘、奇幻森林、冒险营地
- `创想课堂馆`
  - 关键词：手工教室、画纸蜡笔、书本贴纸、创意工作台
- `方块生态馆`
  - 关键词：方块草地、像素树林、立方体生物、生态观察站

统一要求：

- 风格接近首页绘本自然场景
- 不做大面积纯色块
- 不带文字
- 只做封面氛围图，由 HTML 叠加标题与进度

### 2. 普通宠物卡的收集状态

所有宠物卡都保持正常颜色与正常插画显示。

- `已收集`
  - 右上角用“已登记 / 已收录”小印章
  - 可带轻微高亮边框
- `未收集`
  - 保持彩色可见
  - 右上角改成“待发现 / 未登记”小标
  - 可加轻微虚线框、浅雾化纸签、角标，不做整卡灰度

### 3. 二级分类改成“主题册”

全馆状态下：

- 不再把所有 series 一次性平铺
- 只显示一句提示，引导先选择分馆

进入某个分馆后：

- 才显示该分馆下的“主题册”按钮
- 主题册是面向孩子的浏览结构，不直接暴露技术型原始 series

#### 阳光花园馆

- `经典植物册` → `PVZ`
- `写实实验册` → `pvz真实`

#### 奇趣冒险馆

- `神话瑞兽册` → `灵兽族 / 瑞兽族 / 山海族 / 敦煌族`
- `星梦奇旅册` → `星瞳族 / 绮梦族 / 旅行族`
- `萌宠搭档册` → `萌肖族 / 酷肖族 / 绒爪族`
- `守护异兽册` → `守护系 / 双钳族`

#### 创想课堂馆

- `想象涂鸦册` → `萌宠风 / 幻想风`
- `像素机巧册` → `像素风 / 科幻风`
- `东方灵感册` → `国潮风`

#### 方块生态馆

- `原版生态册` → `我的世界`

### 4. 页面信息层次

- 顶部：图鉴馆首页栏目卡
- 中部：4 个分馆封面卡
- 分馆未选中：显示“先选分馆再看主题册”
- 分馆选中后：显示该分馆的主题册按钮
- 下部：宠物卡网格

## 验收标准

- 顶部 4 张分馆卡不再是大纯色块
- 4 张分馆卡视觉风格与首页背景更统一
- 所有宠物卡保持正常彩色
- 未收集宠物不再整体灰掉
- 二级分类不再全量平铺所有 series
- 进入分馆后，主题册按钮与宠物过滤结果一致
- Codex 内置浏览器可实点验证
