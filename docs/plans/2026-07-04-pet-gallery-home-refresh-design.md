# 宠物图鉴分馆首页改版设计

日期：2026-07-04

## 目标

把宠物图鉴集合页从“技术来源分类”改成更自然的“图鉴分馆”体验，同时统一顶层分类名称、按钮图标、来源文案，以及页面顶部总览栏的视觉语言。

## 设计结论

### 1. 顶层分类改成 4 个图鉴分馆

- `阳光花园馆`
  - 收纳：`original` + `pvz`
  - 面向孩子的理解：植物系、花园系、阳光系伙伴
- `奇趣冒险馆`
  - 收纳：`banchong`
  - 面向孩子的理解：冒险旅途中遇到的多族群伙伴
- `创想课堂馆`
  - 收纳：`classpet`
  - 面向孩子的理解：课堂创作、风格练习、主题设定伙伴
- `方块生态馆`
  - 收纳：`minecraft`
  - 面向孩子的理解：方块世界里的生态观察对象

### 2. 顶层分馆卡片统一改成“圆形徽章 + 标题 + 副标题 + 进度”

- 不再使用零散 emoji 作为主图标
- 改用统一圆形徽章按钮图标
- 每个分馆提供一句副标题，帮助孩子理解这个馆收的是什么类型的宠物

### 3. 图鉴页顶部总览区改成“图鉴馆首页栏目卡”

- 左侧主卡：图鉴总览，展示总收集、总体进度
- 右侧栏目卡：稀有度分布
- 右侧栏目卡：当前展区/当前检索状态

### 4. 过滤逻辑同步升级

- 点击分馆卡片后，只展示该分馆下的宠物
- 系列 tab 根据当前分馆动态收窄
- `全部` 仍保留，便于返回全馆总览

### 5. 来源文案统一

- 详情弹层、卡片副标题、图鉴说明中不再直接暴露生硬的技术来源词
- 改为孩子更容易理解的馆名或馆内子说明

## 验收标准

- 集合页顶部不再出现“原生宠物 / 仓鼠大冒险 / 课堂宠物 / 我的世界”这种混杂口径
- `pvz真实` 不再被漏出顶层分类体系
- 顶层 4 卡的图标风格统一，按钮观感更像展馆入口
- 总览区看起来更像图鉴馆首页，不再只是两个普通统计盒子
- 在 Codex 内置浏览器中可实际点击分馆、切换图鉴、查看结果
