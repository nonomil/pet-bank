# docs 目录说明

`docs/` 下面主要是本地方案、调研、会话记录、临时计划和参考材料，不再作为公开远端文档库维护。

公开仓库只保留这个说明文件，避免把大量阶段性草稿、图片、参考材料和临时验收记录继续推送到远端。

公开可读资料请看：

- [项目 README](../README.md)
- [更新日志](../CHANGELOG.md)
- [工程文档索引](../docs_project/README.md)
- [技术架构](../docs_project/ARCHITECTURE.md)
- 本地技术方案：`自托管后端方案/`（不推送远端）

本地仍可保留完整 `docs/` 工作材料；这些文件会被 `.gitignore` 忽略。
