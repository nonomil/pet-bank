## 3. API 契约（Leaderboard —— P1 也依赖此接口，先冻结）

```js
// js/leaderboard.js —— 通用本地排行榜，按 gameId + profileId 隔离
window.Leaderboard = {
  // gameId: 'mathpk' | 'hanzi' | ...；meta 可放 {correct, total, duration} 等
  record(gameId, score, meta = {}),  // 写入，自动维护 best + recent(≤10)，返回 {best, recent}
  getBest(gameId),                    // Number
  getRecent(gameId, n = 10),          // [{score, ts, ...meta}]
  renderUI(containerId, gameId, opts),// 渲染：最高分 + 最近列表 + 进步趋势(简易 sparkline)
};
// 存储 key 规范：petbank_lb_{gameId}_{ProfileManager.getActiveId()||'default'}
// 数据结构：{ best:Number, recent:[{score, ts, ...meta}] }
```

**向后兼容**：P0 做 best-effort 迁移——读取旧 `petbank_math_high_score`，写入 `Leaderboard` 的 mathpk best。

## 4. 题库 Schema（P1 `data/hanzi-questions.json`）

```json
{
  "source": "汉字游乐场·汉字题库", "version": "1.0",
  "levels": {
    "1": [
      {
        "char": "山", "pinyin": "shān",
        "emoji": "⛰️",
        "example": "我们一起去爬**山**。",
        "answer": "山",
        "opts": ["山", "水", "火", "木"],
        "modes": ["choose-char-by-pinyin", "choose-pinyin-by-char", "fill-blank"]
      }
    ],
    "2": []
  }
}
```

- 四要素：`char`/`pinyin`/`emoji`(代图片，零资源成本)/`example`(例句，`**` 标红目标字)。
- `modes`：多种出题模式（看拼音选字 / 看字选拼音 / 例句填空），一卡多用。
- 分级：levels 1-3 起步，子代理各放 8-12 条种子数据。
