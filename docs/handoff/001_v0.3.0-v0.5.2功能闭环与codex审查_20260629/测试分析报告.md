## 6. 验证局限（🔲 待补）

- **未覆盖**：多设备/多浏览器（仅 Chrome + chromium-1187）/ 移动端响应式 / 长时间衰减（decay 跨天）/ localStorage 容量上限
- **盲盒/卡片权重**：用模拟权重逻辑验证（pickRarityItem 闭包私有，未直接调真实 openBlindBox 跑千次统计）
- **数据导出/导入文件 IO**：验证收集/写入逻辑，未真触发文件下载/上传（playwright 文件 IO 复杂）
- **MC 动作图/背景图**：代码接口验证（路径/切换），实际图待用户生（当前 404/渐变兜底）
