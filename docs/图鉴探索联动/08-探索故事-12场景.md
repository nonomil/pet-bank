# 探索故事 - 12 场景第一版文本

> 目标：把现有探索文案统一改写成“图鉴馆委托调查”口径。

## 神秘森林

- 章节：第一章 · 起点花园
- 本章目标：学会观察与登记，让第一批伙伴进入图鉴馆。
- 重点系列：经典植物线 / 萌宠风 / 绒爪族 / 守护系
- 委托任务：图鉴馆委托你进入植物镇外侧的神秘森林，登记最基础也最常陪孩子成长的伙伴。
- 开场文案：你背着空白图鉴册走进森林，树影间传来温室铃声，像是在提醒你第一份记录就要开始了。
- 发现事件文案：在潮湿的树根边，你会捡到写有宠物线索的叶片标签，提示它们平时住在哪、爱做什么。
- 数学事件文案：题目示例：调查册写着“水边有 3 个脚印，花圃有 5 个脚印”，问花圃比水边多几个脚印。能力点是数量比较，答题时要先找到两处线索再相减。
- 选择事件文案：分岔路会让你决定先去水边还是花圃，不同路线会提前遇到不同风格的草木系伙伴。
- 遭遇事件文案：最终遭遇以植物镇基础宠物为主，战斗更像“接受登记考核”，证明你有资格把它写进图鉴。
- 收卡结束语：战斗结束后，图鉴册会自动浮现该宠物的第一页档案，你也正式成为图鉴馆见习调查员。

## 蔚蓝海滩

- 章节：第二章 · 森林边界
- 本章目标：认识旅途型与海风型伙伴，学会从行为判断它们的个性。
- 重点系列：旅行族 / 双钳族 / 酷肖族 / 萌肖族
- 委托任务：图鉴馆要你前往蔚蓝海滩，调查喜欢远行和巡岸的伙伴，补齐“旅行习惯”和“社交方式”两栏。
- 开场文案：海浪把一串写着宠物名字缩写的漂流瓶推到你脚边，说明今天会遇到愿意主动试探你的伙伴。
- 发现事件文案：你会在沙滩、栈桥和贝壳堆里发现它们留下的爪印、羽毛和补给袋。
- 数学事件文案：题目示例：从漂流瓶到栈桥要走 6 步，从栈桥到贝壳堆要走 4 步，先到栈桥再去贝壳堆一共走几步。能力点是路线合计，强调把两段路接起来算。
- 选择事件文案：不同选择会让你先接触双钳族、旅行族或外向型伙伴，影响之后的遭遇顺序。
- 遭遇事件文案：海边的宠物更擅长先观察你，打赢它们后，图鉴会补上“是否愿意和陌生人组队”这一项。
- 收卡结束语：收卡后，海风会吹开图鉴页角，显示这只宠物最喜欢和谁一起远行。

## 糖果王国

- 章节：第二章 · 森林边界
- 本章目标：记录梦境、创意与情绪表达型伙伴，建立图鉴馆的“想象力分馆”。
- 重点系列：幻想风 / 绮梦族 / 萌宠风
- 委托任务：创想课堂馆希望你进入糖果王国，补完那些总被误解成“只会卖萌”的幻想型宠物档案。
- 开场文案：甜雾里漂着会发光的便签纸，上面写着“先看它喜欢做什么，再判断它会不会战斗”。
- 发现事件文案：你会发现糖纸折成的小地图、软糖脚印和梦境碎片，它们都在提示宠物的爱好与专长。
- 数学事件文案：题目示例：梦境配方每份需要 2 颗星星糖，糖罐里有 10 颗，最多能配几份。能力点是平均分组，让“甜点配方”承担真实任务。
- 选择事件文案：选择不同甜点路线，会遇到幻想风、绮梦族和萌宠风的不同代表。
- 遭遇事件文案：这些宠物不一定敌意强，但会用捉迷藏、镜像和错觉考验你是否真的认识它们。
- 收卡结束语：收卡后，图鉴会解锁“喜欢被怎样夸奖”这一栏，让它们不再只是漂亮立绘。

## 彩虹瀑布

- 章节：第三章 · 海边集市
- 本章目标：补完水系、清洁型与照料型伙伴的工作档案。
- 重点系列：写实实验线 / 绒爪族 / 国潮风 / 守护系
- 委托任务：图鉴馆要你调查彩虹瀑布的照料型伙伴，确认它们在队伍里到底承担恢复、清洁还是引路职责。
- 开场文案：瀑布边的雾气里悬着一排湿润标签，每张都写着“工作岗位尚未登记”，提醒你这一章要重点补完它们在队伍里的职责。
- 发现事件文案：你会在石台、水潭和苔藓角落里找到它们的工具、储物袋和清洁痕迹。
- 数学事件文案：题目示例：青蛙伙伴每轮给 3 片荷叶补水，连续 2 轮能补几片荷叶。能力点是乘法意义，把“每轮数量”和“轮数”写成值班记录。
- 选择事件文案：你可以先去浅滩、瀑顶或洞口，不同路线会提前遇到温柔辅助型或谨慎守护型伙伴。
- 遭遇事件文案：它们的战斗更像“上岗考试”，打赢后会解锁“最适合和哪种伙伴组队”这一条。
- 收卡结束语：收卡时，图鉴馆会把这只宠物归入“可靠后勤伙伴”栏目。

## 沙漠绿洲

- 章节：第三章 · 海边集市
- 本章目标：补完远行、敦煌、守望与耐力型伙伴的背景故事。
- 重点系列：敦煌族 / 旅行族 / 山海族
- 委托任务：图鉴馆要你前往沙漠绿洲，记录那些擅长背负、引路与守望的宠物，让它们不再只剩稀有度标签。
- 开场文案：风里夹着半张旧地图，提示你今天登记的不只是能力，还有它们如何穿越孤独与漫长旅程。
- 发现事件文案：你会在驿站残墙、风化石柱和商队脚印之间找到宠物留下的故事线索。
- 数学事件文案：题目示例：商队有 12 瓶水，每只远行伙伴一天需要 3 瓶，能支持几只伙伴一天。能力点是补给分配，答题结果直接影响队伍能否继续前进。
- 选择事件文案：你可以选择跟商队、进壁画洞或追踪风铃，不同选择会先遇到旅行族、敦煌族或山海支线。
- 遭遇事件文案：沙漠宠物出场时更像守路人，打赢它们后才能知道它们究竟是在守护宝物，还是在等同伴归来。
- 收卡结束语：收卡后，图鉴会自动补上一条“它曾经独自走过的路”。

## 珊瑚王国

- 章节：第四章 · 高地洞窟
- 本章目标：补完深水与双钳型伙伴的团队分工与守卫规则。
- 重点系列：双钳族 / 星瞳族 / 我的世界
- 委托任务：珊瑚王国的委托是查清楚哪些伙伴负责巡逻、哪些伙伴负责搬运、哪些伙伴是珊瑚区的守门人。
- 开场文案：水泡里漂着来自图鉴馆的防水札记，提醒你这一次要特别留意“群体协作”字段。
- 发现事件文案：珊瑚缝隙、沉船边和气泡轨迹都会透露宠物的行动规律与职能。
- 数学事件文案：题目示例：巡逻队要求每组 2 条小鱼一起游，8 条小鱼能分成几组。能力点是组队除法，让巡逻阵型成为题目来源。
- 选择事件文案：路线会在珊瑚林、沉船仓和潮汐门之间切换，提前暴露不同宠物族群。
- 遭遇事件文案：海底宠物多半先用阵型试探你，打赢后图鉴会解锁它们“最信任的队友类型”。
- 收卡结束语：收卡时，水面会映出它在珊瑚王国的岗位徽记，让图鉴不只写战力。

## 雪山之巅

- 章节：第四章 · 高地洞窟
- 本章目标：补完灵兽、巡山者与耐寒型伙伴的成长经历。
- 重点系列：灵兽族 / 瑞兽族 / 守护系
- 委托任务：图鉴馆需要你去雪山之巅查证：那些看起来很强的灵兽，到底是天生孤傲，还是长期守山训练的结果。
- 开场文案：第一阵山风吹开图鉴页，露出“童年经历尚未登记”的空栏，暗示这章要更多关注伙伴的成长故事。
- 发现事件文案：冰面抓痕、风铃石和旧巡山旗会提示宠物小时候住在哪里、受过谁的照顾。
- 数学事件文案：题目示例：每段安全绳长 5 米，巡山员把 3 段接在一起，从营地到冰壁共有几米。能力点是等长累加，贴合雪山训练记录。
- 选择事件文案：不同路线会让你先接触灵兽族、瑞兽族，或偏守护型的高地伙伴。
- 遭遇事件文案：山巅宠物擅长把战斗当作资格审查，只有赢过它，图鉴馆才承认你真的理解它的生活方式。
- 收卡结束语：收卡后，图鉴会额外浮现“从小受过哪位前辈影响”。

## 水晶洞窟

- 章节：第四章 · 高地洞窟
- 本章目标：补完隐藏型、守库型与洞窟观察者的档案。
- 重点系列：山海族 / 像素风 / 我的世界
- 委托任务：图鉴馆怀疑许多水晶洞窟里的宠物并不爱主动现身，所以把“观察证据”这一章交给你补全。
- 开场文案：洞口的石牌写着“别急着抓，先学会看”，说明这里的故事要靠耐心和推理推进。
- 发现事件文案：反光矿石、掉落羽片和储物刻痕会提示宠物的作息和领地范围。
- 数学事件文案：题目示例：火把编号按 2、4、6、8 的规律亮起，下一支该点亮几号。能力点是等差规律，需要读懂洞窟机关留下的顺序。
- 选择事件文案：你可以走矿道、回声道或旧仓道，分别对应像素风、山海族和守库型伙伴。
- 遭遇事件文案：洞穴里的宠物更强调“你有没有看懂我留下的线索”，战斗只是最后的确认步骤。
- 收卡结束语：收卡后，图鉴页边会出现它藏起来的那一项真实习惯。

## 云端城堡

- 章节：第四章 · 高地洞窟
- 本章目标：补完守护者、礼仪型与高位稀有宠物的责任档案。
- 重点系列：瑞兽族 / 国潮风 / 守护系 / 科幻风
- 委托任务：云端城堡要求图鉴馆解释清楚：这些看似高冷的宠物为什么要守门、守序或守约。
- 开场文案：你刚走进城门，阶梯尽头就亮起一枚馆徽，提醒你这章要重点登记“责任与岗位”。
- 发现事件文案：徽章、值勤表和城堡回音都会透露宠物在这里承担的秩序工作。
- 数学事件文案：题目示例：城堡有 3 座塔楼，每座安排 2 名守卫，还需要 1 名记录员，一共要安排几名伙伴。能力点是两步计算，符合城堡值班表。
- 选择事件文案：路线会在塔楼、回廊和藏书厅之间切换，对应不同的守护与礼仪型伙伴。
- 遭遇事件文案：城堡宠物更像在审阅你的资格，打赢后才能得到真正的身份说明。
- 收卡结束语：收卡时，图鉴会补出它“为什么选择守护这座城堡”的一句心声。

## 温暖火山

- 章节：第五章 · 星空终点
- 本章目标：补完力量型、锻造型与极限环境伙伴的真实个性。
- 重点系列：酷肖族 / 灵兽族 / 像素风
- 委托任务：图鉴馆要你去火山地带纠正偏见：很多看起来暴躁的宠物，真实工作可能是锻造、照明或护送。
- 开场文案：火山边的热流吹开图鉴馆的红封条，露出“请记录它温柔的一面”。
- 发现事件文案：熔岩脚印、锻造台和热能晶石会提示它们平时如何工作、如何帮助队友。
- 数学事件文案：题目示例：冷却台每轮能降温 4 块火石，连续 3 轮能处理多少块火石。能力点是工作效率乘轮数，和火山工坊流程绑定。
- 选择事件文案：你可以走矿炉、火口边或余温洞，不同路线会遇到酷肖族、灵兽族和像素工坊支线。
- 遭遇事件文案：火山宠物的战斗像一场试火礼，打赢后才能读懂它外热内稳的真正性格。
- 收卡结束语：收卡后，图鉴馆会把它归入“力量型但可托付”的高价值伙伴名单。

## 太空站

- 章节：第五章 · 星空终点
- 本章目标：补完观测型、科幻型与星空分馆核心伙伴的职能档案。
- 重点系列：星瞳族 / 科幻风 / 像素风 / 我的世界
- 委托任务：太空站是星空终点的第一道门，图鉴馆要你补完这里的观测员、导航员和实验型宠物档案。
- 开场文案：舷窗外的星轨和图鉴页上的空白轨道重叠，像是在提醒你：这里记录的是“方向感”和“判断力”。
- 发现事件文案：你会在操控台、星图室和维护廊发现它们留下的观测笔记。
- 数学事件文案：题目示例：星际灯塔按 2、4、8、16 的亮度刻度启动，下一个刻度是多少。能力点是倍增规律，像在核对太空站导航参数。
- 选择事件文案：不同舱段会先遇到星瞳族、科幻风或像素生态伙伴。
- 遭遇事件文案：这里的宠物很重视判断与秩序，打赢后才会把真正的导航数据交给你写入图鉴。
- 收卡结束语：收卡时，图鉴会点亮它专属的星轨徽记。

## 星光花园

- 章节：第五章 · 星空终点
- 本章目标：补完传说级、终章级与图鉴馆压轴伙伴的完整档案。
- 重点系列：瑞兽族 / 星瞳族 / 幻想风 / 灵兽族
- 委托任务：星光花园是图鉴馆最终登记地，只有在前面几章积累了足够记录，花园才会向你展示真正的压轴伙伴。
- 开场文案：你翻开最后一册空白图鉴，花园里的光点自动聚成一句话：‘只记录战斗，是看不见真正伙伴的。’
- 发现事件文案：每一朵星花、每一条光路都在补足宠物的来历、责任与愿望。
- 数学事件文案：题目示例：星花第一圈有 3 颗，第二圈 6 颗，第三圈 9 颗，第四圈应该有几颗。能力点是固定步长规律，用作图鉴馆的毕业考核。
- 选择事件文案：不同路径会提前接触瑞兽族、星瞳族和幻想终章伙伴。
- 遭遇事件文案：这里的宠物更像最终审稿人，它们会确认你是否真的把伙伴写成了有来历、有工作、有性格的人物。
- 收卡结束语：收卡后，图鉴馆总册会补完它的终章档案，你也完成从见习调查员到正式记录员的升级。
