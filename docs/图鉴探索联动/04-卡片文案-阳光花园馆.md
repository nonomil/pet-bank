# 阳光花园馆 卡片文案

> 分馆定位：负责植物镇、温室、实验田一带的图鉴调查，主打基础植物伙伴与写实观察对象。

> 适用范围：图鉴详情页、探索遭遇联动、后续数据接入

## 经典植物线

### 大嘴花

- 图鉴称号：温室惊喜考官
- 副标题：捕捉练习班 · 爆发型伙伴
- 稀有度：普通
- 卡片介绍：大嘴花的第一印象总是吓人一跳，但真正认识后会发现它其实很讲究时机与礼貌。
- 来自哪里：植物镇温室靠里侧的捕捉区
- 小时候：就能一口叼住被风吹跑的训练布，特别喜欢偷偷练习爆发时机。
- 学校 / 学艺：植物镇的捕捉练习班
- 现在的工作：温室里的惊喜考官
- 喜欢做什么：闭眼晒太阳、练习突然出手、观察新人会不会被吓到
- 擅长什么：抓住空隙后突然爆发
- 核心能力：高爆发咬合与时机判断
- 推荐出现场景：神秘森林
- 性格标签：爆发强 / 很会等时机 / 安静 / 反差感大
- 代表技能：时机咬合 / 闭眼感知 / 惊喜考核

大嘴花在植物镇的捕捉练习班很有名，因为它小时候就能一口叼住被风吹跑的训练布。它从小住在温室靠里侧，总爱安静地等到最合适的时候才突然出手。现在它在温室里做惊喜考官，专门负责测试见习调查员有没有认真观察周围。它平时最喜欢闭眼晒太阳，擅长抓住空隙、突然爆发，也因此总被写在“别只看外表”的图鉴备注里。

### 寒冰射手

- 图鉴称号：雾棚降温员
- 副标题：冷雾班毕业生 · 控场型伙伴
- 稀有度：普通
- 卡片介绍：寒冰射手负责让队伍在混乱时重新冷静下来，是植物镇最受欢迎的控场型伙伴之一。
- 来自哪里：植物镇喷雾棚旁的冷雾走廊
- 小时候：最喜欢待在喷雾棚边看水珠凝结，越急的场面越想先让大家冷静下来。
- 学校 / 学艺：植物镇的冷雾班
- 现在的工作：观察温室的降温员
- 喜欢做什么：擦拭雾棚玻璃、记录温度变化、慢慢整理节奏
- 擅长什么：先让局面慢下来，再替队伍争取准备时间
- 核心能力：冷雾减速与节奏回稳
- 推荐出现场景：彩虹瀑布
- 性格标签：冷静 / 控场强 / 节奏感好 / 观察细致
- 代表技能：冷雾减速 / 雾棚巡看 / 清凉回神

寒冰射手在植物镇的冷雾班长大，小时候最喜欢待在喷雾棚边看水珠凝结。它比起一味冲锋，更擅长先让局面慢下来，再替同伴争取整理节奏的机会。现在它在观察温室做降温员，负责处理太热或太急躁的训练场。它喜欢擦拭雾棚玻璃、记录温度变化，也最擅长用冰凉但不刺人的方式让队伍重新找回秩序。

## 写实实验线

### 伴舞僵尸

- 图鉴称号：图鉴记录员 · 叶脉实验员
- 副标题：实景观察课 · 见习试验棚助理
- 稀有度：普通

- 来自哪里：植物镇外侧的写实试验田
- 小时候：跟着研究员辨认不同土壤和光线的差别
- 学校 / 学艺：实景观察课
- 现在的工作：观察温室的见习试验棚助理
- 喜欢做什么：记录天气变化
- 擅长什么：耐心观察细节
- 核心能力：生长修正
- 推荐出现场景：神秘森林
- 性格标签：写实实验线 / 耐心观察细节 / 记录天气变化 / 生长修正
- 代表技能：雾棚恢复 / 神秘森林登记 / 叶面校准

伴舞僵尸出生在植物镇外侧的写实试验田，从小跟着研究员辨认不同土壤和光线的差别。它小时候在实景观察课学习水雾调节，毕业后留在观察温室担任见习试验棚助理。平时它最喜欢记录天气变化，也因此练出了一手耐心观察细节。当冒险者接近神秘森林时，伴舞僵尸往往会主动出来确认对方是不是认真观察过周围环境，所以图鉴馆把它列为这一带最值得登记的伙伴之一。它真正让人佩服的地方，是能把生长修正稳定地用在照顾队伍、守护岗位或完成委托上，而不是只拿来炫耀力量。
