# 叙事重构文档包索引

> 这是一套面向 `pet-bank`（成长伙伴 · 萌宠冒险岛）的新叙事文档包，目标不是把项目做成完整 galgame，而是把“探索层”重设计成更像绘本 / 对话式视觉小说的体验包装。
>
> 本目录强调三件事：
> 1. 项目的主循环仍然是 `积分 -> 宠物 -> 探索 -> 兑换 -> 回到成长`
> 2. 探索故事采用更强的画面感、对话框、立绘和推进节奏
> 3. 数学 PK、道具、场景解锁与剧情事件要自然融入，而不是另起一套大系统

## 3. 设计原则

- 不做完整分支叙事引擎，只做单线主线 + 少量选择。
- 不把探索做成“聊天游戏”，而是做成“有画面、有推进、有奖励”的对话式探索。
- 数学 PK 只作为解谜节点和成长节点，不单独抢主线。
- 首页和探索页都要优先服务“看得懂、点得动、愿意继续看”。
