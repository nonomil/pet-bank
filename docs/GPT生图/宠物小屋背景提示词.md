# 宠物小屋背景生图提示词

> 参考 banchong.cn 宠物小屋的 `room-starter.webp`（1920×1080 房间背景，"初心小屋"）。本项目宠物小屋加背景层，多套主题可切换。用户拿本文档去 chatgpt（GPT-IMAGE-2）生图。

## 背景
banchong 宠物小屋用一张 1920×1080 的房间背景图作为小屋底图（`assets/images/pet/room-starter.webp`，默认"初心小屋"），买不同小屋可换背景（装饰经济）。本项目宠物小屋（`js/home.js`）当前用纯色/渐变背景，加背景层后更美观，且为后续"商店买小屋换背景"预留。

## 统一规范
- **尺寸**：1920×1080（横向，铺满小屋区域）
- **风格**：与本项目深夜主题 + 像素风一致（参考需求规格书 §6 视觉规范）；或温馨水彩绘本（参考 docs/原始需求/UI和场景参考.md）
- **统一风格后缀**：`cozy room interior, soft warm lighting, no characters, no text, no watermark, pixel-art friendly, deep-night palette with accent glow`
- **负面词**：no pet, no character, no text, no watermark, no border（背景里不要宠物/角色，宠物图是叠加的）
- **命名**：`room-{theme}.webp`
- **存放**：`assets/home-bg/`（新建目录）
- **关键**：背景要"留出中央空间"给宠物立绘站（构图：地面/平台居中，四周装饰，不要中央有突兀物）

## 6 套主题背景（每套 1 条完整 prompt）
每条结构：[主题场景描述] + [构图：中央留空给宠物] + [统一风格后缀]

1. **初心小屋（默认）** room-starter.webp —— 温馨木质小屋内景，木地板，小窗透暖光，角落有食盆/玩具，中央留空
2. **森林秘境** room-forest.webp —— 晨光森林空地，蘑菇/藤蔓/树洞，中央草地留空
3. **海洋深处** room-ocean.webp —— 海底珊瑚洞穴，气泡/海草/贝壳，中央沙地留空
4. **星空梦境** room-starry.webp —— 夜空云端，星星/月亮/流星，中央云台留空
5. **糖果乐园** room-candy.webp —— 糖果色甜点世界，蛋糕/棒糖/饼干，中央糖霜平台留空
6. **火山熔岩** room-volcano.webp —— 火山岩洞，熔岩/晶石/暖光，中央岩台留空

（每条补完整英文 prompt，含场景细节 + 中央留空 + 风格后缀。子代理或用户可在此填充完整 prompt。）

## 生图后接入
1. 下载 → 按命名存 `assets/home-bg/room-{theme}.webp`
2. **代码接口已预留**：`js/home.js` 的背景层（`.home-bg` 容器 + `<img>` 或 `background-image`）。默认 `room-starter.webp`，主题切换改 `src`/`background-image`。
3. 后续"商店买小屋"（P2-A 装饰经济，对应 AC-19/20）：商店买 room-forest → home.js 切背景 → localStorage 记 `petbank_home_theme`。

## 风格自洽
背景应与现有 `assets/scenes/`（12 探索场景）+ `assets/banchong/`（宠物立绘）风格协调。建议先生 room-starter（默认）验证风格，再批量。
