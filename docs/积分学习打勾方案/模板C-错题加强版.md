# 模板 C：错题加强版

> 适用：一年级更稳定后 / 家长明确需要错题管理  
> 目标：把每日学习单升级成轻量学习管理页

---

## 1. 设计原则

1. 行级字段更完整
2. 错题从“一个点”升级成“整理块”
3. 更适合小学阶段，不建议作为幼小衔接首版

---

---

## 3. 对应积分建议

当前项目里的实际口径是：

- lesson 正常完成继续沿用原有规则
- 模板 C 当前也不额外发学习单奖励
- 模板 C 的意义是提升错题与复盘记录完整度，而不是提高积分速度

---

## 4. 使用提醒

这版更接近真正的“小学学习管理单”。

优点：

- 统计更完整
- 错题更清楚
- 更适合后续周报

代价：

- 每天填写会更重
- 不适合幼小衔接第一阶段
