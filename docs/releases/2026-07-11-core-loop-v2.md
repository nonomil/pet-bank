# Core Loop v2 发布记录

## 本轮交付

- 统一奖励展示模型：成长分、宠物经验、亲密度、道具、升级和进化。
- 安全奖励反馈卡：使用 DOM 节点和 `textContent`，避免奖励文案注入 HTML。
- 游戏、宝箱、小屋动作接入统一反馈；保留原有 toast/积分兼容逻辑。
- 新增成长日志，最多保留 30 条；小屋显示最近照料和连续照料天数。
- 新增任务完成/撤回/再次完成事件记录，不改变原积分算术。
