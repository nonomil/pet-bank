# Core Loop v1 发布记录

## 范围

本版本把以下链路接通：

`游戏/宝箱奖励 -> 成长积分 + 宠物经验 -> 等级/进化结果 -> 宠物小屋每日照料 -> 下一步推荐`

任务勾选仍保留原有可撤销语义，暂不强行改成不可撤销的奖励事件。
