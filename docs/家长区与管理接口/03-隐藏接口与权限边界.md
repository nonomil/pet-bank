# 03-隐藏接口与权限边界

## 1. 什么叫隐藏接口

这里的隐藏接口指：功能代码还在，但普通页面默认不展示入口。

它适合这些能力：

- 导入数据
- 覆盖本地数据
- 导出诊断
- 清空配置
- 批量同步
- 重置档案
- 迁移或联调工具

这些能力对维护很有用，但对普通孩子和普通家长来说风险太高。

## 2. 当前已隐藏：工具箱数据管理

文件：`js/tools.js`

默认状态下，工具箱只显示：

- `随机点名`
- `番茄计时`

`数据管理` 默认不显示。

## 3. 如何打开高级数据管理

当前有三种方式。

### 方式一：URL 参数

打开：

```text
index.html?parentAdmin=1
```

然后进入 `家长区 -> 工具箱`，会看到 `数据管理`。

### 方式二：本地持久开关

在浏览器控制台执行：

```js
ToolboxSystem.enableAdvancedTools()
```

或：

```js
localStorage.setItem('petbank_parent_admin_tools', '1')
```

之后刷新页面，进入工具箱即可看到 `数据管理`。

关闭：

```js
ToolboxSystem.disableAdvancedTools()
```

### 方式三：本次会话开关

```js
sessionStorage.setItem('petbank_parent_admin_tools', '1')
```

这种方式关闭标签页后失效。

## 4. 这不是安全鉴权

当前隐藏只是前端入口隐藏。  
它能减少误点，但不能当成真正权限系统。

如果未来接入后端角色，应该升级成：

| 角色 | 能看什么 | 不能做什么 |
|---|---|---|
| child | 主导航、宠物、学习、游乐场 | 看不到高级数据、账号、危险操作 |
| parent | 设置、工具、作品、孩子档案 | 默认看不到开发诊断 |
| admin | 数据管理、诊断、迁移、后台巡检 | 必须记录操作日志 |

## 5. 高风险操作规则

后续所有高风险操作都应满足至少三条规则：

- 默认折叠或隐藏。
- 文案明确说明会影响什么数据。
- 执行前必须二次确认。
- 覆盖类操作必须建议先导出备份。
- 云端覆盖本地、本地覆盖云端要分开命名。
- 删除、清空、重置类操作不要用普通绿色按钮。

## 6. 建议继续隐藏或折叠的功能

| 功能 | 当前位置 | 建议 |
|---|---|---|
| 数据导入 | 工具箱高级开关 | 已隐藏，保留 |
| 数据导出 | 工具箱高级开关 | 可隐藏，也可未来放到“备份”卡 |
| 云端覆盖本地 | 设置 / 家庭云端 | 折叠到高级恢复 |
| 清空云端配置 | 设置 / 云端配置 | 保留，但加强说明 |
| 云端诊断 JSON | 设置 / 诊断 | 默认只在联调模式展示 |
| 删除孩子 | 设置 / 孩子档案 | 保留二次确认 |
| 本地重置 | ProfileManager | 不放可见入口 |

## 7. 推荐文案

### 高级入口说明

```text
高级数据工具只用于备份、迁移和故障恢复。导入会覆盖当前本地数据，请先导出备份。
```

### 导入确认

```text
确认导入吗？这会覆盖当前浏览器里的所有 pet-bank 数据。建议先导出备份。此操作不可撤销。
```

### 云端覆盖确认

```text
确认用云端数据覆盖本地档案吗？如果本机有未同步的记录，覆盖后可能丢失。
```
