## 一句话结论

`游乐场` 只放孩子玩的内容。  
`家长区` 放家长才需要看的低频管理入口。  
`数据导入 / 数据覆盖 / 诊断 / 危险操作` 不默认展示，只作为高级家长接口保留。

## 当前已落地

- 顶部右侧新增 `⚙️ 家长区`。
- `成长作品 / 工具箱 / 设置` 已从 `游乐场` 下拉移出。
- `工具箱` 默认只显示 `随机点名 / 番茄计时`。
- `数据管理` 已隐藏，只有 `?parentAdmin=1` 或本地高级开关打开时显示。
- `设置` 已整理为 `账号与孩子 / 家庭云端 / 学习与题目 / 规则模板 / 高级与危险操作` 五个分区。
