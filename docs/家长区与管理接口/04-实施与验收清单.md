## 3. 手工验收

### 普通模式

访问：

```text
http://127.0.0.1:8765/index.html
```

检查：

- 顶部可见 `⚙️ 家长区`。
- 打开 `游乐场` 下拉，不出现 `成长作品 / 工具箱 / 设置`。
- 进入 `家长区 -> 工具箱`，只看到 `随机点名 / 番茄计时`。
- 不应看到 `数据管理`。

### 高级模式

访问：

```text
http://127.0.0.1:8765/index.html?parentAdmin=1
```

检查：

- 进入 `家长区 -> 工具箱`。
- 能看到 `数据管理`。
- 数据管理里有导出和导入。
- 导入前仍会弹确认。

## 4. 后续 P1 建议

### 设置页分区

把设置页整理成 5 个清楚区域：

- `账号与孩子`
- `家庭云端`
- `学习与题目`
- `规则模板`
- `高级与危险操作`

### 规则模板

参考班宠乐园 2 的 `加分/扣分规则管理`，把当前任务项和自定义积分项整理成可编辑模板。

建议先做家庭版模板：

- 阅读完成
- 数学挑战
- 主动复盘
- 整理书包
- 按时睡觉
- 主动帮忙

### 高级与危险操作

统一收纳这些能力：

- 导入 / 导出数据
- 云端覆盖本地
- 清空云端配置
- 导出诊断 JSON
- 本地档案重置

所有危险操作都需要二次确认。

## 5. 不建议做

- 不建议把 `数据管理` 放回普通工具箱。
- 不建议把 `设置` 放进 `游乐场`。
- 不建议给孩子主导航增加 `后台 / 管理 / 数据` 之类入口。
- 不建议用纯前端隐藏当真正权限系统。
- 不建议未登录角色体系前做复杂后台。
