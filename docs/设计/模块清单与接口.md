# 模块清单与接口文档

> 本文档盘点「宠物积分银行」前端 11 个 JS 模块与 4 份数据 JSON 的对外接口、依赖关系、启动接入状态与已知断裂点，为后续修复与联调提供单一事实源。
> 来源映射：源码 `js/*.js`（4264 行）+ `data/*.json`（10505 行）实测提取；需求基线见 [../规格/需求规格书.md](../规格/需求规格书.md)；架构分层见 [./技术架构.md](./技术架构.md)；已知问题归口见 [../路线/差距清单与开发路线图.md](../路线/差距清单与开发路线图.md)。
