- 入口：`奇趣冒险馆 -> 萌爪伙伴册`
- 系列：`萌爪伙伴族`
- source id：`banchong2`
- 图片：每只宠物映射 `6` 阶段，全部本地 `.webp`

这一批不用重抓。后续只需要维护图片完整性、图鉴故事质量和线上路径一致性。

## 2. P0：植物 23 个

推荐定位：`阳光花园馆 -> 甜芽花园册`

植物类素材更适合归入 `阳光花园馆`，与现有 PVZ/植物语义一致，但不要直接混进旧 PVZ 册。它们来自 `班宠乐园2`，风格更偏儿童成长伙伴，应该独立成册。

### 推荐命名

| 类型 | 推荐 | 备选 |
|---|---|---|
| 主题册 | `甜芽花园册` | `花果萌友册`、`软芽伙伴册` |
| 系列名 | `甜芽花园族` | `花果萌芽族` |
| source id | `banchong2_plant` | `banchong2_plants` |
| 来源标签 | `🌱 甜芽花园` | `🌼 花果萌友` |

### 已确认清单

| 类别 | 名称 |
|---|---|
| 花草 | 波斯菊、含羞草、牡丹、蒲公英、向日葵、绣球花 |
| 果蔬 | 草莓、金桔、千禧果、樱桃、榴莲、芒果、苹果、葡萄、桃子、西瓜、香蕉、玉米 |
| 树竹 | 柿子树、银杏树、竹子 |
| 特色植物 | 葫芦、仙人球 |

### 图鉴叙事方向

`甜芽花园册`不建议写成“植物大战”或“农作物大全”，而是写成小朋友可亲近的花果伙伴：

- 草莓：小红帽点心师，负责把练习完成后的奖励做成甜甜贴纸。
- 向日葵：阳光签到员，提醒大家每天先完成一个小目标。
- 竹子：节节高记录员，用一节一节的成长纹路记录努力。
- 含羞草：情绪观察员，适合做“今天有点害羞也没关系”的陪伴角色。
- 银杏树：秋日收藏家，适合承接徽章、成长记录和纪念册。

### 导入策略

植物导入必须慢速、可恢复、可审计：

- 单线程下载，不并发。
- 每张图片之间等待固定间隔加轻微随机抖动。
- 先使用已有只读清单，不重复高频请求 `/api/pets`。
- 每只植物仍按源站 `10` 级映射到当前项目 `6` 阶段：`[1, 2, 4, 6, 8, 10]`。
- 只提交 `.webp`，不提交源站 `.png`。
- 生成独立快照与 manifest，方便以后复跑或回滚。
- 使用唯一 id，例如 `bc2p_strawberry` / `bc2_plant_sunflower`，不能用中文名称做唯一键，因为 `向日葵` 等名称可能和现有 PVZ/旧植物重名。
- 运行时 source 使用 `banchong2_plant`，必要时额外保留 `originSource: "banchong2"` 用于溯源。

建议输出：

- `data/source-snapshots/banchong2-plants.json`
- `data/source-snapshots/banchong2-plants-import-manifest.json`
- `assets/banchong2/甜芽花园族/<名称>-0.webp`
- `assets/banchong2/甜芽花园族/<名称>-1.webp`
- `assets/banchong2/甜芽花园族/<名称>-2.webp`
- `assets/banchong2/甜芽花园族/<名称>-3.webp`
- `assets/banchong2/甜芽花园族/<名称>-4.webp`
- `assets/banchong2/甜芽花园族/<名称>-5.webp`

图片数量口径：

| 策略 | 下载源图 | 最终提交 |
|---|---:|---:|
| 只保留项目 6 阶代表图 | 约 `23 x 6 = 138` 张 | 138 张 `.webp` |
| 保留完整 10 级源档 | 约 `23 x 10 = 230` 张 | 不推荐提交源 `.png` |

## 3. P1：其他/人物 17 个

推荐定位：`锦缘奇遇册`

这组不建议叫“人物册”或“其他册”。源站分类叫 `PERSON`，但实际内容包含历史人物、神话角色、学习主题和道具，产品侧更适合收口成“奇遇”。

### 推荐命名

| 类型 | 推荐 | 备选 |
|---|---|---|
| 主题册 | `锦缘奇遇册` | `云锦传说册`、`瑞灵同游册` |
| 系列名 | `锦缘奇遇族` | `云锦奇遇族` |
| source id | `banchong2_legend` | `banchong2_wonder` |
| 来源标签 | `🎐 锦缘奇遇` | `🏮 云锦传说` |

### 已确认清单

| 类别 | 名称 |
|---|---|
| 历史与文化 | 秦始皇、武则天、诸葛亮、孔子、花木兰 |
| 学习与文昌 | 文昌帝君、文曲星、朝代歌 |
| 神话与角色 | Q版哪吒、Q版孙悟空、Q版猪八戒、财神爷、女魔法师、男魔法师 |
| 奇遇道具 | 赛车、DIY奶茶、上上签 |

图片数量口径：

| 策略 | 下载源图 | 最终提交 |
|---|---:|---:|
| 只保留项目 6 阶代表图 | 约 `17 x 6 = 102` 张 | 102 张 `.webp` |
| 保留完整 10 级源档 | 约 `17 x 10 = 170` 张 | 不推荐提交源 `.png` |

### 产品定位建议

`锦缘奇遇册`可以做成“奇遇伙伴”而不是传统宠物。每个角色不强调被饲养，而强调陪孩子完成一种学习或成长任务：

- 文昌帝君 / 文曲星：学习加油、错题复盘、每日阅读。
- 孔子 / 诸葛亮：思考力、表达力、策略提示。
- 花木兰 / 哪吒：勇气、坚持、挑战关卡。
- 财神爷 / 上上签：奖励、幸运事件、兑换商店彩蛋。
- 朝代歌：中文学习、历史韵律、背诵任务。

## 4. 是否合入现有馆

推荐不新增顶层大馆，继续沿用当前馆结构：

| 内容 | 推荐顶层馆 | 理由 |
|---|---|---|
| 植物 23 个 | `阳光花园馆` | 和植物、自然、花果主题最一致 |
| 其他/人物 17 个 | `创想课堂馆` 或 `奇趣冒险馆` | 如果偏学习陪伴，放课堂；如果偏奇遇伙伴，放冒险 |

当前我更推荐：

- `甜芽花园册` 放 `阳光花园馆`
- `锦缘奇遇册` 放 `创想课堂馆`

这样四馆会更均衡，不会把新增内容继续堆到 `奇趣冒险馆`。
