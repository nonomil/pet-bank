## 3. 宠物小屋（/house）设计要点

- **背景 = 房间图**：默认 `room-starter.webp`「初心小屋」（免费），全屏铺底；切换小屋 = 换房间图。
- **宠物立绘**：大图、透明 PNG，居中（如 `cloud-samoyed-2.webp`，约 400px 宽）。
- **操作区按钮**：`🏠 管理小屋` / `🪆 管理玩具` / `自定义小屋和玩具价格 →` / 徽章墙。
- **串门**：左侧同学列表（去谁家串门），可看同学小屋。
- 引导文案：宠物住简陋初心小屋 → 点「管理小屋」选购温馨专属小屋。

## 4. 「管理小屋」购买弹窗（本项目对齐重点）

- 弹窗标题「小屋管理」+ **余额行**（学生名 · 余额 N 班币）。
- **售楼处**：房间卡片纵向列表（左缩略图 ~80×60 + 右名称加粗 + 描述 + 价格 + 购买按钮）。
- 房间示例：暖阳公寓 30 班币、森林温室 50 班币、…100 班币。
- **班币不足**→购买按钮灰色禁用、卡片轻微变暗、提示「班币不足」。
- **已解锁计数**（X / 总数）。
- 货币：班币（学生做任务赚，教师可「自定义价格」）。

## 5. 本项目对齐情况（已实现）

| banchong | 本项目 | 状态 |
|----------|--------|------|
| 管理小屋弹窗 + 房间网格 | `openManageHome()` 弹窗 + `.home-bg-grid` 8 卡片 | ✅ |
| 余额行 | `#homeManageBalance` 显示成长分 + 已解锁计数 | ✅ |
| 房间描述 | BG_THEMES `desc` 字段 + `.home-bg-desc` | ✅ |
| 班币不足灰色 | `.home-bg-btn.buy.poor`（积分<价格） | ✅ |
| 购买解锁 + 切换 | `buyTheme()` 花分解锁 + `setHomeBg()` 切换 | ✅ |
