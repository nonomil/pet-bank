## 3. 当前统一口径

### 3.1 产品定位

卡牌系统不是独立战役，而是：

**宠物训练营 / 冒险演练**

### 3.2 内容结构

当前推荐口径不是“5章大挑战”，而是：

**5 大区域 / 10 个轻章节**

说明：

- 每个大区域包含 2 个轻章节
- 每个轻章节是 1 次 2 到 5 分钟的短挑战
- 章节强调固定轻奖励，不做刷关收益

### 3.3 积分关系

- 基础练习可玩
- 更好的挑战和奖励，再与训练券 / 积分绑定
- 训练营不应成为主要积分来源
