# 探索 species 敌人 - 正式规则

> 2026-07-01 定稿。探索战斗引入"图鉴宠物 species"作为敌人，与现有 monster 混用。
> 目标：图鉴联动 + 稀有惊喜，**但不让探索变成第二套卡牌主玩法**。

## 一、定位（必须守住）

- **monster**：探索叙事底色（小蜗牛/野兔等），**不变**，负责日常探索氛围与主循环积分产出。
- **species 敌人**：图鉴联动 + 稀有惊喜（包装成 野生/失控/同族守卫/镜像个体）。
- **混用但不反客为主**：探索仍是主循环（产成长积分），species 敌人只是稀有遭遇层。

## 二、适用场景：章末精英区

仅 **danger_level 4-5** 场景遇 species 敌人，标记 `elite: true`：

| danger | 场景 |
|---|---|
| 4 | underwater（珊瑚王国）/ castle（云端城堡）/ space（太空站） |
| 5 | volcano（温暖火山）/ stargarden（星光花园） |

danger 1-3 场景（forest/beach/candy/waterfall/mountain/cave/desert）**仅 monster**，保持低危区纯叙事。

## 三、species 池（按 danger 匹配 rarity）

从 `PetSystem.getAllSpecies()` 按 rarity 过滤（198 只：common 98 / rare 48 / epic 38 / legendary 14，无 uncommon）：

- danger 4 精英区：**epic + legendary**
- danger 5 精英区：**legendary + epic**

## 四、遇敌规则

**elite 场景**每次探索：
- **40%** 概率遇 species 敌人（从该 danger 的 rarity 池随机选 1 只 species）
- **60%** 概率遇 monster（现有 `scene.monsters`）

**非 elite 场景**：100% monster（不变）。

## 五、命名包装（不敌对感）

species 敌人显示名**不直写宠物名**，随机包装（4 选 1）：
- `野生${name}` / `失控${name}` / `${name}同族守卫` / `镜像${name}`

避免"打自己宠物"的敌对感。

## 六、species → 战斗适配

species 转 monster 结构（exploration 1v1 战斗引擎字段兼容）：

```js
{ id, name: 包装名, emoji,
  hp: base_hp, atk: base_atk, def: base_def || 0,
  exp: 10 + danger_level * 5,
  drops: [],
  isSpecies: true, speciesId: species.id, baseName: species.name }
```

## 七、奖励：首胜定向 + 重复降级

**首胜定向**（核心卖点）：
- species 敌人**首次**击败 → **100% 掉该 species 卡**（定向收集：打谁收谁）
- 记录首胜：`localStorage['petbank_species_first_win']` = `[speciesId, ...]`

**重复击败降级**（防刷卡）：
- 已首胜的 species 敌人再次击败 → **10%** 掉该卡 + 常规积分/道具（降级，不稳定掉完整卡）

**monster**（不变）：
- 现有 25% 通用随机掉卡（按 rarity 加权）

整体掉卡率：elite 场景 40%×（首胜 100% / 重复 10%）+ 60%×25% —— 首胜期高、重复期自然回落。

## 八、训练券节奏影响

卡池加速（首胜定向），每日 3 张训练券可能不够打轻章节 → **上线观察**，必要时训练券每日 3 → 4（见 Step6 训练券）。

## 九、改动文件

| 文件 | 改动 |
|---|---|
| `data/scenes.json` | danger 4-5 场景加 `elite: true` |
| `js/exploration.js` | 遇敌（L286）加 species 分支 + species→monster 适配 + 命名包装 |
| `js/app.js` | 掉卡（L1201）加 `isSpecies` 首胜定向 + 重复降级 |

## 十、明确不做

- 不全量替换 `scene.monsters`
- 不每次稳定掉完整卡（首胜后降级到 10%）
- 不在 danger 1-3 场景放 species 敌人
- 不让 species 敌人产出成长积分（探索积分仍走主循环，与 species/monster 无关）
