# 🌱 植物阵营（Plant Cards）

这些是核心输出/防御/功能型植物，适合做主力卡池：

## 🔫 攻击型

- 豌豆射手（Peashooter）
- 双发射手（Repeater）
- 三重射手（Threepeater）
- 寒冰射手（Snow Pea）
- 火炬树桩（Torchwood，偏辅助强化）
- 玉米投手（Kernel-pult）
- 西瓜投手（Melon-pult）
- 卷心菜投手（Cabbage-pult）

## 🛡 防御型

- 坚果墙（Wall-nut）
- 高坚果（Tall-nut）
- 南瓜头（Pumpkin）
- 地刺（Spikeweed）
- 高地刺（Spikerock）

## ⚡ 功能/控制型

- 向日葵（Sunflower，资源核心）
- 双胞向日葵（Twin Sunflower）
- 窝瓜（Squash，爆发清场）
- 樱桃炸弹（Cherry Bomb）
- 土豆地雷（Potato Mine）
- 磁力菇（Magnet-shroom）
- 催眠蘑菇（Hypno-shroom）
- 寒冰菇（Ice-shroom）

## 🌿 特殊/人气扩展（漫画/衍生常见风格）

- 火爆辣椒（Jalapeno）
- 大嘴花（Chomper）
- 食人花升级体（可做“进化卡”）
- 阳光菇（Sun-shroom）

# 🧠 卡牌设计建议（很关键）

你这个玩法可以这样分层：

### 🌱 植物卡

- 资源型（向日葵）
- 输出型（豌豆系）
- 控制型（寒冰/磁力）
- 防御型（坚果）

### 🧟 僵尸卡

- 低费压场（普通/路障）
- 突破防线（撑杆/橄榄球）
- 特殊机制（气球/矿工）
- Boss卡（僵王）

### ⚡ 技能卡（可以参考漫画夸张效果）

- 阳光爆发
- 僵尸潮汐
- 植物进化
- 全场冻结
- 即死爆炸（樱桃炸弹类）
