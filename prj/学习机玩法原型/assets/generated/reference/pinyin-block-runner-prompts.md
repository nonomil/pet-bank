# 拼音块收集台 GPT 生图提示词

生成目标：替换 `拼音贪吃蛇` 的蛇形象，改成无蛇化的“拼音块收集台 / pinyin block runner”。

## 视觉参考图 Prompt

Create one polished visual reference image for a children's keyboard learning mini game called "Pinyin Block Runner".

Game concept: a 6-year-old child uses arrow keys to move a small friendly glowing cursor token across a soft 12x12 learning board, collecting pinyin syllable blocks in order. It should feel like a modern educational arcade board, not a snake game, not a retro handheld console shell. No snake, no worm, no animal body segments, no long trailing body.

Viewport: landscape web game screen, 16:9, suitable for desktop and tablet. The game surface should show:
- a clean rounded square grid board in the center
- a small collectible cursor token or star rover on one tile
- several soft rounded pinyin block tiles placed on grid cells, with blank centers for HTML text overlays
- a top task card area for one Chinese character, pinyin progress chips, and star progress, but leave all text areas blank
- gentle reward elements: stars, progress glow, soft sparkles
- no readable text, no letters, no numbers, no Chinese characters, no pinyin inside the image

Art direction: warm modern children's app UI, soft 3D clay-like rounded tiles, clean shapes, readable game layout, pastel green, mint, yellow, coral, sky blue accents, slightly more mature than toddler toys, appropriate for 6-year-old children. Keep UI simple and playable. Avoid dark arcade, avoid Minecraft style, avoid SVG/vector flat look, avoid oversized toy blobs.

Output: high quality bitmap concept art, no text, no brand logo.

Create a transparent background PNG asset explosion sheet for a children's web game called "Pinyin Block Runner".

Important constraints:
- true transparent alpha background, no checkerboard, no white background

- every component should have its own soft shadow contained inside the object bounds
- no snake, no worm, no animal body segments, no long trailing body

Components to include:
1. rounded 12x12 board panel, blank grid, pastel green, square, complete object
2. small glowing cursor token / star rover, viewed from top, complete object
3. rounded pinyin block tile, blank center for HTML text overlay, mint color
4. rounded pinyin block tile, blank center for HTML text overlay, yellow color

6. rounded pinyin block tile, blank center for HTML text overlay, sky blue color
7. task card plaque, blank center, for Chinese character and pinyin chips overlay
8. star reward badge, no text
9. progress strip, blank, no text
10. small sparkle burst / success effect
11. wrong-answer wobble icon / soft warning puff, no text
12. round arrow-key hint button, blank icon area

Art direction: warm modern children's educational game UI, soft 3D clay-like rounded pieces, clean geometry, suitable for 6-year-old children, lively but not babyish, no brand logo.
