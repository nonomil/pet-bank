# 拼音赛车 GPT 参考图提示词

## Full Reference Prompt

Use case: stylized-concept
Asset type: full visual reference for a browser game prototype
Primary request: Create a polished full-screen landscape reference image for a children's pinyin racing game.
Scene/backdrop: A bright sci-fi toy racing track viewed from a slightly elevated front perspective, designed for a 6-year-old child using a keyboard. The track runs from the bottom center toward the upper middle, with gentle curves, glowing lane lines, speed streaks, coins, stars, and checkpoints. It should feel like a playable game screen, not a marketing poster.
Subject: A small friendly futuristic race car near the bottom center, racing forward. Several blank road-sign target panels float or stand along the track ahead. Each road sign has a large empty area where HTML will later overlay a Chinese character and pinyin. The car should look energetic but not oversized.
Style/medium: High-quality colorful 3D game UI concept art, polished but not too cute, similar to a premium children's arcade learning game. Clean shapes, crisp edges, readable silhouettes, slightly sci-fi, with a fun racing mood.
Composition/framing: 16:9 horizontal game screen. Keep the center track readable and uncluttered. Leave safe blank UI zones at top-left, top-right, and bottom-center for HTML HUD, settings, score, and typing input. Do not cover the car or road signs with decorative clutter.
Lighting/mood: Bright, fast, encouraging, low-pressure. Neon cyan, warm yellow, grass green, and orange accents. No dark horror mood, no heavy battle mood.
Text: No text.
Constraints: Every road sign, UI panel, card, banner, button, license plate, and dashboard must be completely blank. No readable text, no letters, no pinyin, no Chinese characters, no numbers, no fake writing, no logos, no watermark. The image should be useful as a visual design reference for implementing the game in HTML/CSS.
Avoid: No cannons, no missiles, no guns, no explosions as the main theme, no brand names, no copyrighted characters, no dense tiny details, no over-cute mascot taking over the screen.
