# Transparent Asset Sheet Prompt

Create one transparent PNG asset sheet for the same original children's typing mini game.

Output:
- true transparent background with alpha channel
- no checkerboard pattern
- no colored canvas
- no text, no numbers, no letters, no pinyin, no Chinese characters
- isolated reusable game components with generous transparent spacing
- shadows must stay inside each component bounds

Components:

1. original blocky green boom monster, idle front view
2. same monster walking toward the player, feet planted on a shared ground baseline, four blocky feet in left-right alternating walking poses
3. same monster defeated into green cube particles
4. blank rounded target bubble
5. large bottom keyboard input panel, blank
6. red heart health icon
7. empty heart health icon
8. gold star reward icon
9. small green cube particle burst
10. soft warning impact flash

Style:
- original voxel-inspired children's game art
- friendly, cute, readable at small size
- not Minecraft Creeper, not a copyrighted character
- no scene, no sky, no grass, no full screen UI mockup
- character feet must align to one baseline so the webpage can scale from the feet without floating
