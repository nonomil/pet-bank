# ☀️ 暑假成长积分银行

> 一个为 7 岁孩子量身定制的「成长积分银行」——把暑假目标变成可触摸、可积累、可炫耀的小小成就。游戏化 + 长期主义，**让孩子在暑假中找到成长的乐趣**。

## ✨ 设计理念

**"不懂代码的爸爸"也能给孩子做的好东西。**

把"完成目标"包装成"赚取积分"，把"坚持习惯"变成"积累财富"，把"自律"变成一种**看得见、摸得着、能炫耀的小小成就系统**。

## 🎯 五大能量维度

| 维度 | 能量色 | 核心目标 | 示例任务 |
|------|--------|----------|----------|
| 📚 **学习力** | 蓝色 | 暑假作业、阅读打卡 | 完成数学练习 +10分，阅读30分钟 +8分 |
| 🏃 **运动力** | 橙色 | 每日运动、户外活动 | 跳绳100下 +10分，游泳30分钟 +15分 |
| 🎮 **自控力** | 绿色 | 屏幕时间管理、早起 | 屏幕时间≤1小时 +10分，7点前起床 +8分 |
| 🔬 **探索力** | 紫色 | 科学实验、新技能学习 | 做一个科学实验 +15分，学一首新曲 +12分 |
| 🛠️ **实践力** | 红色 | 家务劳动、手工制作 | 整理房间 +8分，洗碗 +6分 |

## 🏆 奖励兑换体系

积分可兑换不同级别的奖励：

| 等级 | 所需积分 | 奖励示例 |
|------|----------|----------|
| ⭐ 青铜 | 50 | 多看15分钟动画片 |
| ⭐⭐ 白银 | 150 | 周末去公园/博物馆 |
| ⭐⭐⭐ 黄金 | 300 | 选一个心仪的玩具 |
| 👑 钻石 | 500 | 一天" immunity day"（免做家务） |

## 📊 功能页面

1. **成长地图** — 5个维度关卡节点，可视化进度
2. **今日打卡** — 每日任务清单，勾选即得积分
3. **每周复盘** — 周度统计图表，趋势分析
4. **奖励兑换** — 积分商城，选心仪奖品
5. **成长作品** — 上传展示孩子的作品
